## 1. PROTEGER ENDPOINTS COM orderId

Localizar endpoints como:

- GET /api/nfe/order/:orderId/info
- GET /api/nfe/order/:orderId/*
- qualquer endpoint que use req.params.orderId

---

Antes de qualquer acesso a dados:

Adicionar:

ts const orderId = Number(req.params.orderId); await validateOrderTenant(orderId); 

---

## 2. PROTEGER ENDPOINTS COM :id (NF-e por ID)

Localizar endpoints como:

- GET /api/nfe/:id
- GET /api/nfe/xml/:id
- GET /api/nfe/danfe/:id
- GET /api/nfe/download/:id

---

⚠️ IMPORTANTE: nesses casos o ID NÃO é orderId

---

Antes de retornar dados:

Adicionar:

ts const nfe = await storage.getNFe(req.params.id);  if (nfe?.orderId) {   await validateOrderTenant(nfe.orderId); } 

---

👉 NÃO alterar retorno existente  
👉 NÃO mover lógica  
👉 apenas inserir validação após leitura inicial

---

## 3. (OPCIONAL MAIS SEGURO)
