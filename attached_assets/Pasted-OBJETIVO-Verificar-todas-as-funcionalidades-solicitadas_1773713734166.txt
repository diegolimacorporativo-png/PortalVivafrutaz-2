OBJETIVO

Verificar todas as funcionalidades solicitadas anteriormente e garantir que todas foram implementadas corretamente.

Se alguma funcionalidade ainda não existir no sistema ela deve ser criada automaticamente.

Se existirem pastas, serviços ou rotas sem interface visual, o sistema deve gerar automaticamente as páginas de frontend correspondentes.

Após implementar o sistema deve executar testes visuais automáticos e gravar vídeos comprovando o funcionamento.
