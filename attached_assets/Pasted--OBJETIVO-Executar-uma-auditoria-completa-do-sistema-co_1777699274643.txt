   - páginas admin

   - integrações com backend

## 🌦️ PARTE 2 — BUG IA CLIMA (CRÍTICO)

### PROBLEMA

IA não retorna clima em tempo real

---

## 🛠️ INVESTIGAR

1. Onde a IA de clima está implementada:
   - service de IA
   - endpoint
   - integração externa (API clima)

2. Verificar:

- Está usando API real ou mock?
- Está com chave inválida?
- Está sem await?
- Está retornando cache antigo?
- Está com erro silencioso (try/catch vazio)?

---

## 🔧 CORRIGIR

Aplicar correção mínima necessária:

- Garantir chamada real (ex: OpenWeather, WeatherAPI, etc.)
- Garantir async/await correto
- Garantir retorno estruturado:

```ts
{
  cidade: string,
  temperatura: number,
  descricao: string,
  atualizadoEm: string
}
