ETAPA 2.1 — CLASSIFICAÇÃO COMPLETA DAS 101 TABELAS
========================================================

Classifique TODAS as tabelas public em:

1. TENANT_SCOPED
Dados pertencem obrigatoriamente a uma empresa.

Ex:
orders
products
deliveries
financial_transactions

2. GLOBAL_SHARED
Dados globais compartilhados entre tenants.

Ex:
planos
modulos_sistema

3. SYSTEM_INTERNAL
Infraestrutura interna do sistema.

Ex:
audit_logs
workflow_events
cron_alert_logs

4. PUBLIC_READ
Dados públicos intencionais.

Ex:
about_us

5. SENSITIVE_CRITICAL
Dados críticos que exigirão RLS rigoroso.

Ex:
users
financial
contracts
nf-e

Entregar:
- tabela
- classificação
- justificativa
- tenant key atual
- risco atual
- recomendação futura
