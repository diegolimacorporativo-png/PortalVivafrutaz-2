# 🧨 PROBLEMAS IDENTIFICADOS (BUGS SILENCIOSOS)

---

## 1. Risco de XML inválido por caracteres especiais

Campos como:

- xNome
- xFant
- xProd
- natOp

Podem conter:

- &
- <
- >
- "

👉 Pode quebrar XML silenciosamente

---

## 2. Sanitização duplicada/inconsistente

Hoje existe:

- sanitizeXml()
- xmlSafe()

👉 Pode gerar comportamento inconsistente

---

## 3. Números inválidos ainda podem escapar

Mesmo com validação:

- valores podem virar string inválida no XML
- risco de "NaN", "Infinity"

---

## 4. Campo opcional indo como string vazia

Exemplo:

<IE></IE>

👉 Alguns validadores fiscais rejeitam isso

---
