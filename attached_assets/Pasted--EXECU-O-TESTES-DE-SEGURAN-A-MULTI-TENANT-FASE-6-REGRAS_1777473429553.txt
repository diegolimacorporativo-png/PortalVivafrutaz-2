# 📦 ETAPA 1 — EXECUTAR TESTE HTTP (READ)

Rodar:

bash npx tsx scripts/test-tenant-http-security.ts 

---

## 📊 VALIDAR RESULTADO

Para cada cenário:

### ✔ Esperado

- Mesmo tenant → 200  
- Outro tenant → 403  
- Sem auth → 401  

---

## ❌ ERRO CRÍTICO (ABORT)

Se aparecer qualquer:

- 200 em cross-tenant ❌  
- 500 ❌  
- vazamento de dados ❌  

👉 PARAR IMEDIATAMENTE

---

# 📦 ETAPA 2 — EXECUTAR TESTE WRITE

bash npx tsx scripts/test-tenant-write-security.ts 

---

## 📊 VALIDAR

- Escrita cross-tenant → BLOQUEADA  
- Sem bypass via endpoints POST/PUT
