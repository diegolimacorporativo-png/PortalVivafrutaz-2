# 🛠️ ETAPA 1 — VALIDAR TOTAIS

Verificar:

- Soma de todos vProd dos itens = total da NF (vNF)
- vNF deve ser igual a:
  vProd + vFrete + vSeg - vDesc

Se divergente:

❌ ERRO: TOTAL_MISMATCH

---

# 🛠️ ETAPA 2 — VALIDAR ITENS

Para cada produto:

- qCom > 0
- vUnCom > 0
- vProd = qCom * vUnCom (2 casas)

Se inválido:

❌ ERRO: ITEM_INVALID

---

# 🛠️ ETAPA 3 — VALIDAR ICMS (SIMPLES NACIONAL)

Verificar:

- CRT = 1
- CSOSN presente (ex: 102, 400)
- NÃO deve existir campos de ICMS incompatíveis (ex: CST com CRT=1)

Se inválido:

❌ ERRO: ICMS_INVALID

---

# 🛠️ ETAPA 4 — VALIDAR CFOP

Verificar coerência básica:

- 5102 → venda dentro do estado  
- 6102 → venda fora do estado  

Se inconsistência detectada:

⚠️ ALERTA: CFOP_POSSIBLY_INVALID

(NÃO bloquear — apenas alertar)

---

# 🛠️ ETAPA 5 — VALIDAR CAMPOS OBRIGATÓRIOS

Emitente:

- CNPJ
- xNome
- endereço completo

Destinatário:

- xNome
- município
- UF

Produtos:

- xProd
- NCM
- CFOP

Se faltar:

❌ ERRO: MISSING_REQUIRED_FIELD

---

# 🛠️ ETAPA 6 — VALIDAR ESTRUTURA DO XML

Verificar:

- contém <NFe>
- contém <infNFe>
- contém <det>
- contém <total>

Se inválido:

❌ ERRO: XML_STRUCTURE_INVALID

---

# 🛠️ ETAPA 7 — DETECTAR DADOS INVÁLIDOS

Procurar no XML:

- "NaN"
- "undefined"
- valores vazios em tags críticas

Se encontrado:

❌ ERRO: INVALID_CONTENT

---

# 🛠️ ETAPA 8 — RESULTADO FINAL

Gerar resumo:

- totalItens
- valorTotal
- quantidade de erros
- quantidade de alertas

---

# 📊 CLASSIFICAÇÃO FINAL

- ✅ APROVADO → nenhum erro crítico
- ⚠️ APROVADO COM ALERTAS → apenas CFOP ou warnings
- ❌ REPROVADO → qualquer erro crítico
