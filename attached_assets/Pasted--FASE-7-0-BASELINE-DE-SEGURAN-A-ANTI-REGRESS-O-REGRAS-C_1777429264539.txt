# 🔁 ETAPA 3 — CENÁRIOS

Testar cada endpoint em:

---

### 1️⃣ SUCESSO

✔ usuário válido  
✔ tenant correto  

---

### 2️⃣ SEM AUTH

✔ sem cookie/token  

Esperado:
- 401

---

### 3️⃣ CROSS-TENANT

✔ usuário de outra empresa  

Esperado:
- 403  
- log [SECURITY] TENANT_MISMATCH  

---

### 4️⃣ NOT FOUND

✔ ID inexistente  

Esperado:
- 404
