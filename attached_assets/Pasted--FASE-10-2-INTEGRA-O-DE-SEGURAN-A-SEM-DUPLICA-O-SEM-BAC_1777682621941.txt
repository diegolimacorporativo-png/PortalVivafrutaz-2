---

## 🛠️ ETAPA 1 — ADICIONAR RESUMO DE CONTAS BLOQUEADAS

Dentro de security-intelligence.tsx

Adicionar nova query:

ts const { data: lockedAccounts } = useQuery({   queryKey: ['/api/security/locked-accounts'], }); 

---

Criar um CARD simples no topo:

tsx <Card>   <CardHeader>     <CardTitle>Contas Bloqueadas</CardTitle>   </CardHeader>   <CardContent>     <p>{lockedAccounts?.length ?? 0} contas bloqueadas</p>     <Button onClick={() => navigate('/admin/system-health?tab=locked')}>       Ver detalhes     </Button>   </CardContent> </Card> 

---

## 🛠️ ETAPA 2 — LINKAR PARA SYSTEM HEALTH

👉 NÃO recriar tabela  
👉 NÃO copiar código  

Apenas navegação:

ts navigate('/admin/system-health?tab=locked')

## 🛠️ ETAPA 3 — ADICIONAR RESUMO DE LOGS

ts const { data: securityEvents } = useQuery({   queryKey: ['/api/admin/security/events'], });

tsx <Card>   <CardHeader>     <CardTitle>Eventos de Segurança</CardTitle>   </CardHeader>   <CardContent>     <p>{securityEvents?.length ?? 0} eventos registrados</p>     <Button onClick={() => navigate('/admin/security-audit')}>       Ver logs     </Button>   </CardContent> </Card>
