📊 2. FUNÇÃO PRINCIPAL

Usa:

const data = await authReadService.getAuthAttempts({
  from: last24h
});

---

🔍 3. TIPOS DE ANOMALIAS

## 3.1 BRUTE FORCE DISTRIBUÍDO

Detectar quando:

- múltiplos IPs atacam a mesma conta
- ou 1 IP ataca múltiplas contas

Sinal:
score +30

---

## 3.2 LOGIN SPIKE

Detectar:

- aumento súbito de falhas vs média histórica

Comparação:
- last 1h vs last 7d baseline

Sinal:
score +25

---

## 3.3 IP VOLATILITY

Detectar:

- mesma conta logando de muitos IPs em pouco tempo

Sinal:
score +20

---

## 3.4 DEVICE ANOMALY (se existir deviceId)

Detectar:

- mudança de device em múltiplos logins consecutivos

Sinal:
score +15

---

## 3.5 FAILURE CLUSTERING

Detectar:

- tentativas agrupadas em janelas curtas (<5 min)

Sinal:
score +10

---

📊 4. OUTPUT DO ENGINE

Retornar:

{
  generatedAt: string,
  window: "24h",
  globalRiskSignal: number (0–100),
  anomalies: [
    {
      type: "BRUTE_FORCE | SPIKE | IP_VOLATILITY | DEVICE | CLUSTER",
      severity: "low | medium | high",
      score: number,
      affectedEntities: string[],
      evidence: object
    }
  ]
}
