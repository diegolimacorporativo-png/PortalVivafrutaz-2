### 1. Descobrir schema

sql

FROM pg_tables
WHERE tablename = 'nfe_emissoes'
ORDER BY schemaname;

### 2. Validação obrigatória

- >1 linha → ❌ PARAR (ambiguidade)

### 3. Estrutura

sql

FROM information_schema.columns
WHERE table_name = 'nfe_emissoes'

### 4. Total de registros

### 5. NULL em order_id

### 6. Status existentes

### 7. Constraints

sql
