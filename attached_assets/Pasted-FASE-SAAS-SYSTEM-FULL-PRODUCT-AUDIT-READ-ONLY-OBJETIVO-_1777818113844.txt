========================================================
1. PLANOS (PLANS)
========================================================

Verificar:

- existe tabela de planos?
- campos:
  ✔ nome
  ✔ preço
  ✔ limites (ex: pedidos, usuários, empresas)
  ✔ features (flags)

- planos são:
  - fixos no código?
  - dinâmicos no banco?

- existe plano padrão/free?

========================================================
2. ASSINATURA (SUBSCRIPTIONS)
========================================================

- existe vínculo empresa → plano?
- campos:
  ✔ status (active, trial, canceled)
  ✔ startedAt
  ✔ expiresAt
  ✔ trial?

- sistema reconhece plano atual em runtime?

========================================================
3. CONTROLE DE LIMITES
========================================================

- criação de pedidos além do limite
- número de usuários
- uso de features premium

IMPORTANTE:

- limite é:
  - apenas visual?
