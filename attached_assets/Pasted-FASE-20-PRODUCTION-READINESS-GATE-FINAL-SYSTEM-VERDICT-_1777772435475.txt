  - divergência CLI vs sistema

C) stabilityScore
- baseado em:
  - chaos test results
  - session stability
  - NF-e concurrency

D) architectureScore
- baseado em:
  - duplicações detectadas
  - N+1 queries

E) observabilityScore
- baseado em:
  - logging coverage
  - audit completeness
  - traceability

========================================================
5. CÁLCULO FINAL

function calculateProductionScore(scores) {

  return (
    scores.securityScore * 0.30 +
    scores.consistencyScore * 0.20 +
    scores.stabilityScore * 0.25 +
    scores.architectureScore * 0.15 +
    scores.observabilityScore * 0.10
  )
}
