## 🧱 PADRÃO ARQUITETURAL

Cada domínio segue:

txt server/routes/<domain>.routes.ts 

Com padrão:

ts export function register(app: Express) {   // handlers copiados 100% iguais } 

E no routes.ts:

ts import { register as domainRegister } from "./domain.routes";  domainRegister(app);
