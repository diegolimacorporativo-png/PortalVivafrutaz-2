STEP 3.5 — PRICE RESOLVER (CORE)

Criar:

server/modules/products/utils/priceResolver.ts

---

FUNÇÃO PRINCIPAL:

export function resolveProductPrice(params: {
  basePrice: number;
  subCategoryPrice?: number | null;
  priceGroupMarkup?: number | null; // ex: 12 (%)
  contractPrice?: number | null;
}): number

---

REGRA DE PRIORIDADE (OBRIGATÓRIA):

1️⃣ contractPrice (MAIOR prioridade)
2️⃣ subCategoryPrice
3️⃣ basePrice

Após escolher o preço base:

4️⃣ aplicar priceGroupMarkup (se existir)

---

IMPLEMENTAÇÃO:

export function resolveProductPrice({
  basePrice,
  subCategoryPrice,
  priceGroupMarkup,
  contractPrice,
}: {
  basePrice: number;
  subCategoryPrice?: number | null;
  priceGroupMarkup?: number | null;
  contractPrice?: number | null;
}): number {
  let price = basePrice;

  if (contractPrice != null) {
    price = contractPrice;
  } else if (subCategoryPrice != null) {
    price = subCategoryPrice;
  }

  if (priceGroupMarkup != null) {
    price = price * (1 + priceGroupMarkup / 100);
  }

  return Number(price.toFixed(2));
}
