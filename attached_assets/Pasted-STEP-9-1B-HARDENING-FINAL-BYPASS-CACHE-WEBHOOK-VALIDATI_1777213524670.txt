## STEP 2 — CACHE NO USO (PERFORMANCE)

Arquivo:
onde está computeUsageAndLimits

---

### OBJETIVO:
Evitar múltiplos cálculos pesados por requisição

---

### IMPLEMENTAÇÃO:

Adicionar no topo do arquivo:

ts
const usageCache = new Map();

---

Dentro da função:

ts
export async function computeUsageAndLimits(companyId) {
  const cached = usageCache.get(companyId);

  if (cached && Date.now() - cached.timestamp < 30000) {
    return cached.data;
  }

  const data = await realComputation(companyId);

  usageCache.set(companyId, {
    data,
    timestamp: Date.now(),
  });

  return data;
}

---

### REGRAS:

✔ TTL = 30 segundos  
✔ NÃO alterar lógica de cálculo  
✔ apenas cache em memória
