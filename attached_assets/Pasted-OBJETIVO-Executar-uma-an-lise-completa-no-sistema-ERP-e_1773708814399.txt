CORREÇÃO DE COMPONENTES SELECT

Erro identificado:

<Select.Item /> must have a value prop that is not an empty string.

Esse erro ocorre quando um SelectItem possui:

value=""

ou

value={""}

Escanear todo frontend.

Diretórios:

pages
components
modules
billing
subscriptions
admin

Localizar:

Select
SelectItem
Select.Item

Substituir automaticamente.

EXEMPLO

ANTES

<SelectItem value="">
Plano Básico
</SelectItem>

DEPOIS

<SelectItem value="plano_basico">
Plano Básico
</SelectItem>

Garantir que cada Select possua:

<SelectValue placeholder="Selecione uma opção" />
