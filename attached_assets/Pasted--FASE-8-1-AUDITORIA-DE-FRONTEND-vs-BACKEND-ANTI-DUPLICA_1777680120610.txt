---

## 🛠️ ETAPA 1 — MAPEAR BACKEND (ROTAS)

📍 Procurar em:

server/routes/  
server/modules/**/routes  
routes.ts  

---

### Para cada endpoint encontrado, montar:

ts {   path: "/api/...",   method: "GET | POST | PUT | DELETE",   categoria: "auth | orders | nfe | finance | security | admin | logistics | etc",   protegido: true | false,   tipoAuth: "requireAuth | requireSessionOrCompany | público", } 

---

## 🛠️ ETAPA 2 — MAPEAR FRONTEND

📍 Procurar em:

client/src/pages/  
client/src/components/  
client/src/hooks/  

---

### Para cada tela:

ts {   page: "nome-da-tela",   endpointsConsumidos: ["/api/..."], } 

---

## 🛠️ ETAPA 3 — CRUZAMENTO (O MAIS IMPORTANTE)

Criar tabela:

| Endpoint | Método | Tem UI? | Tela | Status |
|----------|--------|--------|------|--------|

---

### Classificação:

- ✅ USADO → tem frontend
- ❌ NÃO USADO → sem frontend
- ⚠️ PARCIAL → frontend incompleto
