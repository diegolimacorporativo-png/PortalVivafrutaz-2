📦 TAREFAS

---

1. CRIAR ESTADO DE FILTRO

Arquivo:
client/src/pages/admin/orders.tsx

Adicionar no topo do componente:

const [paymentFilter, setPaymentFilter] = useState<"all" | "paid" | "pending">("all");

---

2. ADICIONAR CHIPS DE FILTRO

Local:
Acima da tabela de pedidos (onde já existem filtros, se houver)

Adicionar:

<div className="flex gap-2 mb-3">
  <button
    onClick={() => setPaymentFilter("all")}
    className={paymentFilter === "all" ? "chip-active" : "chip"}
  >
    Todos
  </button>

  <button
    onClick={() => setPaymentFilter("paid")}
    className={paymentFilter === "paid" ? "chip-active" : "chip"}
  >
    Pagos
  </button>

  <button
    onClick={() => setPaymentFilter("pending")}
    className={paymentFilter === "pending" ? "chip-active" : "chip"}
  >
    Pendentes
  </button>
</div>

⚠️ Reutilizar estilos existentes (não criar sistema novo)

---

3. APLICAR FILTRO NA LISTAGEM

Local:
Antes do .map() dos pedidos

Substituir:

orders.map(...)

por:

orders
  .filter((order) => {
    const isPaid = (order as any).isPaid;

    if (paymentFilter === "paid") return isPaid === true;
    if (paymentFilter === "pending") return !isPaid;

    return true;
  })
  .map(...)

---

4. COMPORTAMENTO

Cenário	Resultado
"default"	Todos os pedidos aparecem
"Pagos"	Só pedidos com isPaid === true
"Pendentes"	Só pedidos com isPaid === false
Troca de filtro	instantânea (client-side)
Sem reload	✔
Sem nova API	✔
